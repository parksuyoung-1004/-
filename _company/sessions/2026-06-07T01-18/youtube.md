# 📺 레오 — 구체적이고 실행 가능한 한국어 지시

📊 채널 분석을 하려면 YouTube API 키와 채널 ID를 설정해야 합니다.  이 단계를 거쳐야 합니다.

가장 먼저 채널 분석의 주요 요구사항(구독자 수, 조회수, 시청 지속률 등)을 이해해야 합니다. 이후에 각 에이전트에게 필요한 데이터 획득 경로와 수행할 작업 목록을 만들어야 합니다.

**필요한 정보:**

1.  **채널 ID:** 현재 유효한 YouTube 채널 ID를 알아야 합니다. 유튜브에서 검색하여 확인하세요. (필수)
2.  **YouTube API 키:** 유튜브 계정에 연결된 YouTube API 키입니다. 유튜브에서 설정해야 합니다. (필수)
3. **유튜브 채널 ID:** 분석할 특정 채널의 ID입니다(예: channel_id = 1234567890).  분석할 채널을 선택하세요.

**다음 단계를 위한 자세한 지침:**

1.  **채널 분석 목표 정의**: 어떤 정보를 얻고 싶은가? (예: 구독자 규모, 평균 시청 지속률, 최근 영상 템플릿 등)
2.  **에이전트 역할 및 책임 분배**: 각 에이전트에게 어떤 역할을 부여해야 하는지 결정합니다. 각 에이전트에 필요한 데이터 획득 경로와 수행할 작업 목록을 구성합니다.
3.  **데이터 추출 경로 정의:** 유튜브 채널에서 필요한 데이터를 추출하는 방법을 계획합니다 (예: YouTube Data API 사용, 웹 스크래핑 등).

**추가적으로 고려해야 할 사항:**

*   **유튜브 데이터 API 사용 방법**: 유튜브 데이터 API를 사용하는 방법을 익히세요.
*   **API 키 보안**: API 키를 안전하게 관리하고 비밀번호로 저장해야 합니다 (API Key를 잊어버리면 서비스 이용 불가).
*   **정기적인 검토**: 채널 분석 결과를 정기적으로 검토하여 전략을 업데이트합니다.

이 가이드라인을 통해 유튜브 채널 분석을 효율적으로 수행하고, 목표 달성을 위한 기반을 마련할 수 있습니다.
